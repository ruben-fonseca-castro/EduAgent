Foundations of Probability

Probability theory is the backbone of many computer science applications, from machine learning to network reliability. Let’s start with the basics: **sample spaces** and **events**. A sample space is the set of all possible outcomes of an experiment, while an event is a subset of these outcomes. For example, in a spam filtering system, the sample space could be all possible emails, and an event could be the set of emails marked as spam.

Axioms of Probability

Probability follows three fundamental axioms:

  1. Non-negativity: \( P(E) \geq 0 \) for any event \( E \).
  2. Normalization: \( P(S) = 1 \), where \( S \) is the sample space.
  3. Additivity: For mutually exclusive events \( E_1 \) and \( E_2 \), \( P(E_1 \cup E_2) = P(E_1) + P(E_2) \).

Conditional Probability and Bayes’ Theorem

Conditional probability \( P(A|B) \) is the probability of event \( A \) given that event \( B \) has occurred. Bayes’ Theorem is a powerful tool for inverting conditional probabilities:

$$ P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)} $$

For instance, in fault detection systems, Bayes’ Theorem helps calculate the probability of a system failure given certain symptoms.

Key Concept: Probability is a measure of uncertainty, and understanding its axioms and conditional forms is crucial for analyzing randomized systems.

Random Variables and Distributions

Random variables represent outcomes of random experiments numerically. They can be **discrete** (e.g., number of packet losses) or **continuous** (e.g., time to process a request). The **Probability Mass Function (PMF)** describes the probability of discrete outcomes, while the **Cumulative Distribution Function (CDF)** gives the probability of a value being less than or equal to a threshold.

Key Distributions

  • Bernoulli: Models a single trial with success probability \( p \).
  • Binomial: Models \( n \) independent Bernoulli trials with success probability \( p \).
  • Geometric: Models the number of trials until the first success.

Expectation and Variance

The **expectation** (or mean) of a random variable \( X \) is \( E[X] = \sum x \cdot P(X=x) \). The **variance** measures the spread of the distribution: \( \text{Var}(X) = E[(X - E[X])^2] \).

Simulating Stochastic Processes

Stochastic processes involve randomness over time. Monte Carlo simulations use random sampling to model these processes. For example, estimating \( \pi \) or analyzing randomized algorithms like quicksort.

Monte Carlo Estimation of \( \pi \)


import random

def estimate_pi(num_samples):
    inside_circle = 0
    for _ in range(num_samples):
        x, y = random.random(), random.random()
        if x**2 + y**2 <= 1:
            inside_circle += 1
    return 4 * inside_circle / num_samples
